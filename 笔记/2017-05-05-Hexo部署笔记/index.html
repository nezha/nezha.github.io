关于公式的问题

主要是说：原生的hexo主题没有MathJax解析器，我们需要配置后才能解析latex格式的公式。

  • 安装hexo-math插件
  • 当然还会有类似下划线的markdown解析冲突

这个就采取最简单的方式出现_，就用\_去解析

关于基本的MathJax公式使用及详细问题可以参考这个文献：LaTeX 在 Hexo 中的使用 -hexo-math
